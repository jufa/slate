# Night Timelapse synchronization slate for phone
This offline app is sync'd to phone clock and is designed to be photographed by a night timelapse camera in order to help calibrate timelapse time, location, and direction. This addresses the issue where aurora photographers may not have the camera clock set accurately, or remeber the direction and location that a timelapse was captured.

To photograph a phone screen with a camera that is set up for night timelapse, close down the aperture to increase depth of field. Hold the phone about 2-4 meters from the camera very steadily.

demo: [https://auroreye.ca/slate](https://auroreye.ca/slate)

run in repositry directory:
`python http.server 8080`
or
`npx http-server -p 8080`
then navigate to 
`localhost:8080/slate.html`

To save as a home screen icon on a phone, use option 'add to homescreen' in your phone's OS
